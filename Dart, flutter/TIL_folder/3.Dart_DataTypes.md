# **DataTypes**

~~~dart
void main() {
  String str = "str"; //String
  bool alive = true; // boolean
  int age = 10; //integer
  double money = 69.99; //double

  num x = 10; // num이 integer의 부모 클래스
  x = 10.11; // num이 double의 부모 클래스 
}
~~~

String : 문자 타입<br>
bool : 참 거짓 (true, false)<br>
int : 숫자 <br>
double : 숫자지만 소수점까지<br>

알아두면 좋은점! : <br>
num은 int와 double의 부모이다
